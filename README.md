# BCA's Voice Agent

BCA's Voice Agent is a desktop-based AI assistant developed using Python. The application integrates Ollama's local Large Language Model (TinyLlama) with a graphical user interface built using Tkinter. It supports real-time response streaming and text-to-speech functionality, allowing users to interact with the AI through a simple and responsive interface.

## Features

* Real-time AI response streaming using Ollama
* Text-to-Speech (TTS) support using pyttsx3
* Multiple voice selection options
* Stop speaking functionality
* Multithreaded architecture for smooth performance
* User-friendly Tkinter GUI
* Local AI processing without relying on cloud services
* Clear chat functionality

## Technologies Used

* Python
* Tkinter
* Ollama
* TinyLlama
* pyttsx3
* Requests
* Threading
* Queue Management

## Project Objective

The objective of this project is to create a lightweight desktop AI assistant capable of generating responses from a locally hosted language model and delivering them through both text and voice output. The project demonstrates the integration of artificial intelligence, graphical user interfaces, multithreading, and speech technologies in Python.
